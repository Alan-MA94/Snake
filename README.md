# 🐍 Slightly Bugged PySnake Game

A classic Snake game implementation in Python, with a few intentional quirks to keep things interesting! Built with modular components for easy maintenance and features.

## 🎮 Features

- **Classic snake mechanics** - Eat food, grow longer, avoid walls and yourself
- **Score tracking** - Keep track of your high scores
- **Modular design** - Separate classes for snake, food, and score management
- **Clean code structure** - Easy to understand and modify

## 🐛 Known "Features" (Bugs)

This game comes with some charming quirks:
- The snake might have occasional existential crises (it was created by a Turtle, so ... no wondering)
- Food items sometimes question their life choices
- Score calculations may use snake or Turtle math (approximately correct)
- Collision detection has a philosophical approach to boundaries

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- No additional dependencies required!
